# Earthquake Risk Prediction System 🌍

## 📌 Overview
This project is a community-focused system designed to predict earthquake risk levels using machine learning techniques. It helps improve disaster preparedness and public safety.

## 🚀 Features
- Predicts earthquake risk using ML models
- Classifies regions into Low, Medium, High risk
- Provides safety guidelines for users
- Visualizes risk data

## 🛠️ Tech Stack
- Python
- Machine Learning (Random Forest / Linear Regression)
- Pandas, NumPy
- Flask / Streamlit (optional)

## 🌍 Community Impact
- Helps people understand earthquake-prone areas
- Supports disaster management planning
- Raises awareness about safety precautions
