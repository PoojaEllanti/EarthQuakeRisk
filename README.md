# 🌍 Earthquake Risk Prediction System

## 📌 Overview

This project is a community-focused system designed to predict earthquake risk levels using machine learning techniques. It helps improve disaster preparedness and public safety by analyzing environmental and seismic factors.

---

## 🚀 Features

* Predicts earthquake risk using machine learning models
* Classifies regions into Low, Medium, and High risk
* Provides safety guidelines for users
* Visualizes risk data for better understanding

---

## 🛠️ Tech Stack

* Python
* Machine Learning (Random Forest / Linear Regression)
* Pandas, NumPy
* Flask / Streamlit (optional for UI)

---

## 🌍 Community Impact

* Helps people identify earthquake-prone areas
* Supports disaster management and planning
* Raises awareness about safety precautions
* Enables better preparedness to reduce potential damage

---

## ▶️ How to Run

```bash
pip install -r requirements.txt
python app.py
```

---

## 🔥 Conclusion

This project focuses on solving a real-world problem by combining machine learning with disaster management, making it highly relevant and impactful for community safety.
